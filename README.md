# 📊 Customer Churn Prediction Dashboard

A modern web application for predicting customer churn using Machine Learning and Streamlit.

---

## 🚀 Features

- Customer Churn Prediction
- AI-powered Prediction Model
- Confidence Score
- Risk Level
- Customer Summary
- Recommended Actions
- Interactive Dashboard
- Plotly Charts

---

## 🛠 Technologies Used

- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- Plotly
- Joblib
